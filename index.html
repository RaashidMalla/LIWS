Read-only paths used by LWIS.

MySQL configC:\xampp\mysql\bin\my.ini

          
mysqladminC:\xampp\mysql\bin\mysqladmin.exe

          
Apache binaryC:\xampp\apache\bin\httpd.exe

          
Apache configC:\xampp\apache\conf\httpd.conf

          
MySQL userroot (no password)

        

        

          
About

          

            LWIS shuts MySQL down with mysqladmin shutdown on every app close —
            including Windows shutdown — so InnoDB never gets corrupted from a hard kill.
